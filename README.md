<p align="center">
  <b>Full-stack movie ticket booking web application</b><br/>

</p>

<p align="center">
  🔗 <a href="https://bookyourticket.vercel.app/" target="_blank"><b>Live Demo</b></a>

---


## 🚀 Features

- User authentication (signup & login)
- Browse movies and show timings
- Seat selection and ticket booking
- Responsive UI (desktop & mobile)
- REST API based backend

---

## 🛠 Tech Stack

**Frontend**
- React.js
- HTML, CSS, JavaScript
- Axios

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Deployment**
- Frontend: Vercel
- Backend: API server
